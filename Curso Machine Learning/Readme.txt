Este repositorio contiene la solucion del Reto de Machine Learning asignado el cual fue "Siguiente Sucursal DHL en Nuevo Leon" y 
los entregables finales del Curso de Inteligencia Artificial.

A continuacion se explicaran los contenidos de las carpetas en este repositorio.

-Reto:
Contiene la solucion del reto. Dentro de esta carpeta se encuentra una carpeta llamada DHL La cual contiene
bases de datos y el codigo del reto el cual es denominado "definitivo".

-E1 Portafolio de Analisis:
Portafolio de Análisis, en esta entrega se integran los reportes y análisis realizados en los diferentes módulos 
sobre los cuales ya se ha tenido retroalimentación. Se espera que se entreguen las correcciones con la retroalimentación de las etapas previas.

Este portafolio contiene las carpetas 
Big Data (Big Data Challengue)
Estadistica avanzada para la ciencia de datos (Venta de Televisores y Link del codigo)
Reto (Video de la solucion del Reto y Documento)

-E2 Portafolio de Implementacion:
Portafolio de Implementación, en esta entrega se integran los proyectos realizados en los diferentes módulos,
 sobre los cuales ya se ha tenido retroalimentación previamente, se espera que se entreguen las correcciones sugeridas en las etapas de retroalimentación previas.

Este portafolio contiene las carpetas

Estadística avanzada para la ciencia de datos (Procesamiento de Datos Multivariados o "Los peces y el Mercurio" y Link del codigo)
Herramientas del Lenguaje Natural (ChatBot de Los Tacos el socio)
Tecnicas y arquitectura de Deep Learning (Trabajos realizados de Deep Learning)
